
从日志中可以看出核心问题是`Access denied to: http://repo.spring.io/plugins-release/org/antlr/com.springsource.antlr/2.7.7/com.springsource.antlr-2.7.7.pom`，直接在浏览器中访问<http://repo.spring.io/plugins-release/org/antlr/com.springsource.antlr/2.7.7/com.springsource.antlr-2.7.7.pom>，显示`HTTP is not supported. Please use HTTPS`，那根源问题就清楚了，肯定是<http://repo.spring.io>已经升级为https了，所以在pom.xml中使用https进行访问即可:

```xml
  <repositories>
    <repository>
      <id>Spring Repository</id>
      <url>https://repo.spring.io/plugins-release/</url>
    </repository>
  </repositories>
```
