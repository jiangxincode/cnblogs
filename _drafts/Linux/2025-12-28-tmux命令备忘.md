# tmux命令备忘

## 常用命令

```Bash
tmux new -s aloys
tmux detach #断开当前会话
tmux a -t aloys
tmux kill-session -t aloys
tmux attach -t aloys 2>/dev/null || tmux new -s aloys #自定连接或者创建session
tmux list-session # 查看所有会话
tmux ls # 查看所有会话，提倡使用简写形式
```

要想复制粘贴，需要选用鼠标选中要粘贴复制的文本，先不要松开鼠标，按住Shift，然后松开鼠标，然后右键。

## 快捷键

### session

```Plain
d：分离当前会话。
s：列出所有会话。
$：重命名当前会话。
```

### window

```Plain
Ctrl+B：
c #创建新窗口
, #给新窗口命名
数字键（如 0-9） # 切换到指定编号的窗口
n # 切换到下一个窗口
p # 切换到上一个窗口
l # 切换到最后使用的窗口
& # 输入 y 确认关闭窗口
```

### Panel

```Plain
" # 水平分屏
% # 垂直分屏
方向键（↑ ↓ ← →） # 按方向键切换到相邻面板
o # 轮转切换面板（按顺序循环）
q # 显示面板编号，按编号切换（如输入 0-9）
x #输入 y 确认关闭面板
```

### 其它

```Plain
[ # 通过鼠标方式滚动屏幕
ESC/q # 退出
```
